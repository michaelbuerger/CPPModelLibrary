# CPPModelLibrary
Ideals: Fast, Cross-Platform, Easy to use with OpenGL and Direct3D
#### Model formats to support reading from and writing to (likely in this order):
##### Priority:
- ASCII OBJ
- STL
- Collada
- FBX
- 3DS

##### Later:
- IGES
- STEP
- VRML/X3D
